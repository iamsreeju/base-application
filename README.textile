h1. My Base

Uses Rails 4, Ruby 2.0, MySQL, Twitter Bootstrapper, RailsAdmin, Devise, Cancan, Rolify etc.

h2. Ruby on Rails

This application requires:

* Ruby version 2.0.0
* Rails version 4.0.0

h2. Database

This application uses MySQL with ActiveRecord.

h2. Development

* Template Engine: ERB
* Testing Framework: RSpec and Factory Girl
* Front-end Framework: Twitter Bootstrap (Sass)
* Form Builder: SimpleForm
* Authentication: Devise with google, twitter and facebook connect
* Authorization: CanCan

h2. Email

The application is configured to send email using a Gmail account.

h2. Configure

Refer config/application.example.yml and set your environment variables with config/application.yml
